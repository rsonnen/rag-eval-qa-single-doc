# Changelog

## [1.0.0] - 2026-01-08

### Added

- Initial public release
- Single document Q/A generation with LangGraph ReAct agent
- Batch corpus processing with SQLite-based resume
- Semantic deduplication to prevent equivalent questions
- Validation with separate LLM model to prevent self-confirmation
- Support for PDF, Markdown, XML/JATS, HTML, AsciiDoc, and plain text documents
- Textual and visual generation modes
- OpenTelemetry tracing support
- Configurable models via config.yaml
